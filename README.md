# 📚 Online Book Store SQL Analysis Project

A comprehensive PostgreSQL project focused on designing and analyzing an Online Book Store database. This project demonstrates real-world SQL skills through database creation, data import, data modeling, and business-driven analytics.

## 🚀 Project Highlights

✅ Designed and managed a relational database in PostgreSQL

✅ Imported and processed data from multiple CSV files

✅ Performed sales, customer, and inventory analysis using SQL

✅ Generated business insights through advanced SQL queries

✅ Applied Joins, Aggregate Functions, Subqueries, GROUP BY, HAVING, and Window Functions

## 📂 Dataset

- Books.csv
- Customers.csv
- Orders.csv

## 📊 Key Business Questions Solved

- Which books generate the highest revenue?
- What are the best-selling genres?
- Who are the top customers by purchase value?
- Which books need inventory replenishment?
- What are the monthly sales trends?
- Which authors contribute most to overall sales?

## 🛠️ Skills Demonstrated

- PostgreSQL
- Database Design
- Data Cleaning
- Data Analysis
- Business Intelligence
- SQL Query Optimization
- Reporting & Insights Generation

## 📈 Project Outcome

Developed an end-to-end SQL analytics solution that transforms raw bookstore data into actionable business insights, helping stakeholders make informed decisions regarding sales, inventory management, and customer engagement.

⭐ Ideal project for showcasing SQL, PostgreSQL, and Data Analytics skills in a Data Analyst portfolio.
